## Introduction

schoolm8 is a tool packed with features for power users and ordinary users alike.

## Customization

### Visual editor

Customize every rounded corner, every color and every shadow with the visual editor, available in the customization section of settings. The visual editor is designed to be simple and easy to use, while still providing a high level of customization.

More available here: [Customization](/docs/users/customization)

### Code editor

The code editor is designed for customizing the raw classes of the page bypassing any limitations of a visual editor, and is only recommended for power users.
The code editor is available in the customization section of settings, and is also used for creating unique community themes that set them apart.

View more info on the code editor and customizations [here](/docs/developers/customization#code-editor)
